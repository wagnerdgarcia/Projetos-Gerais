## Tags

**<!doctype html>** Define o Site como HTML 5

## Principais TAGS

**\<head>** ⇒ Cabeçalho dos Sites
**\<body>** ⇒ Corpo dos Sites

### Tags do Head

**\<title>** ⇒ Titulo do Site
**\<meta>** ⇒ Meta dados do site

### Tags do Body

**\<h>** ⇒ Título
**\<p>** ⇒ Paragrafo
**\<hr>** ⇒ Linha

#### Tags de Formataçao de texto

**\<strong>\<b>** ⇒  <strong> Negrito </strong>
**\<em>\<i>** ⇒ <em>Italico</em>
**\<u>** ⇒ <u>Underline</u>
**\<strike>** ⇒ <strike>Tachado</strike>

#### Tags de Criação de Listas

**\<ul>** ⇒ Criação de listas não ordenadas
<ul>
    <li> <strong>< li ></strong> ⇒ Itens das listas não ordenadas
</ul>

**\<ol>** ⇒ Criação de listas ordenadas

<ol type="1" >
    <li> <strong>< li ></strong> ⇒ Itens das listas ordenadas
    <li> <strong>type</strong> ⇒ Tipo do contador
    <ul>
        <li> 1 ⇒ numeral
        <li> A ⇒ Letra Maiuscula
        <li> a ⇒ Letra Minuscula
        <li> I ⇒ Numero Romano Maiusculo
        <li> i ⇒ Numero Romano Minusculo
    </ul>
</ol>

#### Tags para Imagens

**\<img src =" ">** ⇒ tag que insere uma imagem
<ul>
    <li> <strong>src</strong> ⇒ Caminho da imagem a ser inserida
    <li> <strong>width</strong> ⇒ Largura da imagem
    <li> <strong>heigth</strong> ⇒ Altura da imagem
</ul>

#### Tags para Link

**\<a href = " " >** ⇒ Criação de links entre paginas
<ul>
    <li> <strong>href</strong> ⇒ Caminho da pagina a ser aberta 
</ul>

#### Tags para Tabelas

<ul>
    <li>< table > ⇒ Cria a Tabela
    <ul>
        <li> border ⇒ borda da tabela 
        <li> width ⇒ Largura da tabela
        <li> heigth ⇒ Altura da tabela
    </ul>
    <li> < th > ⇒ Linha de Título 
    <li> < tr > ⇒ Linhas da tabela 
    <ul>
        <li> rowspan ⇒ Agrupa celulas na culuna da tabela 
    </ul>
    <li> < td > ⇒ Colunas da tabela 
    <ul>
          <li> colspan ⇒ Agrupa celulas na linha da tabela 
    </ul>
</ul>

#### Tags para Formulario

<ul>
    <li>< form > ⇒ Cria o formulario
    <ul>
    </ul>
    <li> < input > ⇒ Cria um campo para o formulario
    <ul>
        <li> type ⇒ Define o tipo do campo 
        <ul>
            <li>text
            <li>button
            <li>password
            <li>submit
            <li>radio
            <li>checkbox
        </ul>
        <li> name ⇒ define o nome daquele campo 
        <li> value ⇒ valor do campo 
    </ul>
</ul>
